General Commands

Commands any user can use.

  • /AFK

    Notify users you're not currently at your computer.

    Arguments: List, Set

  • /Bot

    View information about GirBot.

    Arguments: Changelogs, Donate, Feedback, Info, Links, Owner, Ping, Support, Uptime, Vote

  • /Music

    Play music in voice chats.

    Arguments: Bassboost, Clear, Loop, Lyrics, Pause, Play, Playing, Previous, Queue, Remove, Resume, Seek, Shuffle, Skip, Skipto, Stop, Volume

  • /Notepad

    Save notes in a notepad.

    Arguments: Add, Delete, Edit, Notes

  • /Search

    Search the web.

    Arguments: Bing, Corona, Crypto, Ddg, Docs, Github, Google, Hexcolour, Itunes, Npm, Steam, Translate, Weather, Youtube

  • /Serverstats

    View stats about the server you're in.

    Arguments: Animated-emoji, Boosts, Bots, Channels, Emoji, Members, News-Channel, Roles, Stage-Channels, Static-Emojis, Text-Channels, Tier, Time, Voice-Channels
