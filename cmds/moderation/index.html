Moderation Commands

Commands mods & admins can use.

  • /Announcement

    Generate an embedded announcement.

    Arguments: Create, Edit

  • /Automod

    Default server protection commands.

    Arguments: Add, Antiinvite, Antilinks, Antispam, Display, Linkschannel, Remove

  • /Autosetup

    Setup channels for specific activities.

    Arguments: Customvoice, Fun, Games, Logs, Ticketpanel, Welcome

  • /Config

    Custom messages for users.

    Arguments: Leavemessage, Levelmessage, Levels, Setchannelname, Setcolor, Setverify, Ticketmessage, Welcomemessage

  • /Customcommands

    Allows admins to make custom commands.

    Arguments: Add, Delete

  • /Guild

    View information about your server.

    Arguments: Channelinfo, Emojis, Info, Inviteinfo, Members, Oldestmember, Roleinfo, Stealemoji, Userinfo, Youngestmember

    • /Serverstats

    View stats about the server you're in.

    Arguments: Animated-emoji, Boosts, Bots, Channels, Emoji, Members, News-Channel, Roles, Stage-Channels, Static-Emojis, Text-Channels, Tier, Time, Voice-Channels

  • /Invites

    Manage user invites.

    Arguments: Add, Leaderboard, Remove, Show

  • /Levels

    Manage user levels.

    Arguments: Createreward, Deletereward, Leaderboard, Rank, Rewards, Setlevel, Setxp

  • /Messages

    Manage user messages.

    Arguments: Add, Createreward, Deletereward, Leaderboard, Remove, Rewards, Show

  • /Moderation

    Moderate users.

    Arguments: Ban, Banlist, Clear, Clearuser, Demote, Kick, Lock, Lockdown, Nuke, Softban, Tempban, Timeout, Unban, Unlock, Unwarn, Warn, Warnings

  • /Setup

    Easily setup some simple chats for your members.

    Arguments: Customvoice, Deletesetup, Fun, Games, Logs, Ticketpanel, Tickets, Welcomechannels, Welcomeroles

  • /Stickymessages

    Sticky a message in any chat.

    Arguments: Messages, Stick, Unstick

  • /Tickets

    Create and modify tickets.

    Arguments: Add, Claim, Close, Create, Delete, Information, Lower, Notice, Open, Raise, Remove, Rename, Transcript, Unclaim

  • /Voice

    Manage server voice chats.

    Arguments: Limit, Lock, Rename, Unlock
